About This Project

Purpose

This site provides analysis of publicly released documents from the Jeffrey Epstein investigation. The goal is to make these documents accessible and searchable, with every finding linked to source documents for independent verification.

Methodology

1. Data Extraction

All ZIP archives were extracted using standard decompression tools. Total dataset: 18,550 files, 19.43 GB.

2. Text Extraction

PDF text was extracted using PyMuPDF (fitz library). For scanned documents, Tesseract OCR was used.

3. Name Extraction

Names were extracted using pattern matching. Initial extraction found 65,488 unique strings, which were filtered to 23 key individuals after removing locations, dates, and metadata artifacts.

4. Redaction Analysis

All PDFs were scanned for potential redaction bypasses. The Masseuse List was confirmed to have proper redactions (names fully removed). The Contact Book intentionally shows names with only personal contact information redacted.

5. Verification

Every finding links to the source document. Users are encouraged to verify claims independently.

Tools Used

  • Python 3.13 - Primary scripting language
  • PyMuPDF - PDF text extraction
  • Tesseract OCR - Optical character recognition for scanned documents
  • PowerShell / WSL - Archive extraction

Limitations

  • Automated name extraction may miss some individuals or include false positives
  • OCR accuracy varies by document quality
  • Some documents are scanned at low resolution
  • Redacted content cannot be recovered if properly applied

Adding New Files

This site is designed to accommodate future document releases. To add new files:

  1. Extract new archives to the extracted/ directory
  2. Run the extraction scripts to process new PDFs
  3. Update the JSON data files in data/
  4. Commit and push to GitHub for automatic deployment
