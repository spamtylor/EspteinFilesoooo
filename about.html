About This Project

Purpose

This project provides analysis and organization of publicly released documents from the Jeffrey Epstein federal investigation. The goal is to make this data more accessible and searchable while maintaining links to original source documents for verification.

This is not affiliated with the U.S. Department of Justice or any government agency.

Methodology

1. Data Extraction

ZIP archives from DOJ releases were extracted, yielding 18,550 files across multiple formats (PDF, JPG, MP4, DAT, etc.).

2. PDF Text Extraction

Text was extracted from 14,700 PDFs using PyMuPDF. Files without embedded text were processed with OCR (Tesseract).

3. Entity Extraction

Natural language processing was used to identify person names mentioned in documents. These were then manually curated to remove false positives.

4. Categorization

Individuals were categorized based on their documented roles: co-conspirators, victims, associates, legal professionals, judges, law enforcement, and government officials.

Limitations

  • OCR Accuracy: Text extracted via OCR may contain errors from low-quality scans.
  • Name Extraction: Automated name extraction may miss individuals or misidentify names.
  • Redactions: Many documents contain redactions. Redacted content is not accessible.
  • Completeness: This represents publicly released documents only.
